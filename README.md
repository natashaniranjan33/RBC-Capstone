# RBC-Capstone
Wealth Management Client Prioritization Framework for Advisors


This project, developed with RBC, builds a multi-model machine learning framework that translates client behavioral data into a clear advisor prioritization system. Three predictive models — portfolio segmentation, money movement classification, and beneficiary conversion scoring — work together to identify clients at risk of outflow, clients ready to consolidate external wealth, and non-clients already embedded in RBC households who are primed to convert. Across all three, the framework surfaces nearly $1.6 billion in at-risk and capturable assets and approximately 3,600 future clients already within RBC's existing network.

The framework makes visible what routine reporting misses. It flags 455 genuine outflow-risk clients ($817M at stake), identifies 643 consolidation opportunities ($789M in external AUM), and surfaces ~3,600 priority beneficiary prospects already inside existing RBC households. 



The data already exists. The roadmap didn't. By applying machine learning across three connected workstreams, this project moves from raw behavioral data to a structured, advisor-ready prioritization framework, one that distinguishes clients who need retention attention, clients positioned to deepen their relationship with RBC, and non-clients already close enough to convert.
The scale of the opportunity is significant. Across retention risk, consolidation opportunity, and beneficiary conversion, the framework surfaces nearly $1.6 billion in combined at-risk and capturable assets, plus 1,585 future clients already embedded in RBC's existing network.
Built to be used. The framework is grounded in observed behavioral patterns across 30,200 active clients and over 107,000 non-client parties, validated on held-out test data. The models perform with high confidence wherein outflow and consolidation flags carry over 99% model certainty. Every client has a signal. Every signal has a recommended approach. RBC already has the relationships, the data, and the advisors. This framework connects all three.
